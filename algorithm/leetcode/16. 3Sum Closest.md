## 16. 3Sum Closest

一个数组有 n 个整数，额外有一个目标整数，要在数组中找到三个值相加最接近目标整数，返回这三个整数的和。
需要假设每个输入只有一个特定的答案。

## Example:

给出数组 = [-1, 2, 1, -4], 目标 = 1.

最接近的目标的和是 2. (-1 + 2 + 1 = 2).

## 解答思路

- 对数组所有可能的排列求和、过滤、排序，找到最接近目标的

## Tips

- 数组可能很长
- 可以使用 Set 来进行数组的去重
- 使用了 O(n^3) 的思路比较笨重

## 查看别人答案的优秀思路

- 首先对数组排序，对 i 一次遍历，
- 使用 while(j<k)
- sum 比 target 小，j 继续向后
- sum 比 targe 大，k 继续向前
- 比如 [1,2,3,4,10]，目标是 8，最后是 a1,b3,c4
- 比如 [1,2,4,8,16], 目标是14，最后是 a2,b4,c8
- 这样总能找到指定的值
